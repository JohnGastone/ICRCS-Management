Type scale — Inter

Body sits at 15px / 1.6. The scale is custom (note 15px base, not Tailwind's 16px). Load Inter explicitly — the source app references it but never links the webfont, so it silently falls back to system-ui.

Single source of truth. Tokens live in src/theme/icrcs-theme.js (Tailwind 3, imported into tailwind.config.js) and mirror to src/theme/icrcs-theme.css as an @theme block for Tailwind 4. Change a value in one, change it in the other — then class names resolve identically across every portal. Base styles and utilities (.card-shadow, .animate-fade-in, .min-touch-target) live in the global index.css.
